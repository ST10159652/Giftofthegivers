# Giftofthegivers

Disaster Management System

Overview

This web application is designed to facilitate disaster management efforts by providing a platform for incident reporting, task management, and donation tracking. The system allows users to register, log in, and access various features to support disaster response and recovery efforts.

Features

- User Registration and Login: Users can create an account and log in to access the system's features.
- Home: A dashboard providing an overview of ongoing incidents, tasks, and donations.
- Incident Reports: A page for reporting and tracking incidents, including details such as location, severity, and status.
- Donations: A page for tracking donations, including monetary and in-kind contributions.
- All Tasks: A page listing all available tasks, including task descriptions, due dates, and assignment status.
- My Assignments: A page displaying tasks assigned to the logged-in user, including due dates and task status.
- Create Task: A feature allowing authorized users to create new tasks and assign them to users.
- My Profile: A page displaying user profile information, including name, contact details, and task history.

Technical Requirements

- Front-end: HTML, CSS, JavaScript (with a framework such as React or Angular)
- Back-end: Server-side language (such as Node.js, Python, or Ruby) with a database management system (such as MySQL or MongoDB)
- Authentication: User authentication and authorization system

Installation and Setup

1. Clone the repository using Git.
2. Install dependencies using a package manager (such as npm or pip).
3. Configure the database and server settings.
4. Run the application using a local development server.

Usage

1. Register for an account and log in to access the system's features.
2. Report incidents and track their status.
3. View and claim available tasks.
4. Manage donations and track their impact.
5. Update your profile information and view your task history.
